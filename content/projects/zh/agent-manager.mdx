Agent Manager 针对的是一个非常具体的工作方式：我会同时运行很多 Claude Code Agent，但这些会话分散在不同 workspace 中，管理起来很混乱。

## 目标

我希望有一个真正好用的界面，可以把每个 workspace 下的 Agent 组织起来，并在一个地方看到它们的运行状态、任务进度和可操作入口。

## 重点不是堆功能

这个项目的重点不在于把终端简单搬到图形界面里，而是判断哪些信息应该被抽出来做统一管理，哪些又应该继续留在原生环境里。

## 目前在做什么

目前主要工作是深度调研现有解决方案，并判断到底是否需要自己 DIY 一套，还是在现有生态上做更轻的扩展层。
