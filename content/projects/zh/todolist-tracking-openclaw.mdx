TodoList Tracking OpenClaw 想把传统待办工具“只能记录，不能推进”的问题往前推一步：不仅追踪任务，还要让 Agent 参与执行和提醒。

## 目标方向

- 统一来自不同输入媒介的任务
- 自动对任务做分类与优先级判断
- 能自己完成的任务尽量自动执行
- 做不了的任务再回到人类审批与补充
- 用 dashboard 展示项目状态与待处理事项

## 为什么值得做

对个人开发者来说，真正消耗精力的不是写下一条 todo，而是后续的跟进、切换和记忆恢复。这个项目希望把那部分认知负担交给 Agent。

## 当前状态

目前还在定义输入方式、任务分类粒度和 UI 形态，但方向已经明确：它应该是一个 AI-native 的任务管理系统，而不是传统清单工具的轻微升级。
