AIHost is a product for live event hosting. The goal is to turn the whole chain of script writing, line splitting, voice generation, and live playback into a usable workflow.

## Main capabilities

- Generate editable host scripts from flowcharts or documents
- Split scripts into manageable script lines
- Generate audio for each line and manage it as an asset
- Support batch generation, manual edits, and auto-broadcast modes
- Enhance transition lines based on live context

## More than TTS

What matters here is not wrapping a speech API. The project tries to let AI play an actual host role in real event operations.

## Current state

It has gone through several iterations already, moving from MVP-level script handling and voice generation toward auto-broadcasting, state detection, and more agentic live coordination.
