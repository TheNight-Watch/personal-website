LifeCoach is an attempt to unify a personal knowledge system, task system, schedule manager, and long-term memory layer.

## Structure

It is less a single tool than a personal operating system. Inputs can start as fleeting notes, clippings, todos, reviews, or long-term goals, and the system keeps reorganizing them into something more stable over time.

## What it tries to solve

- Move fragmented input into durable structure
- Connect tasks, time, and goals instead of storing them separately
- Use regular review as a source of memory
- Give AI a more stable personal context layer

## Focus

The real question is not how many views it has, but whether the structure can stay maintainable over time and actually help thought and action.
